# 🎬 Concept

The goal of this project was to create a luxury-level emotional web experience that feels personal, warm, and unforgettable.

Instead of building a traditional greeting page, the website was designed like a cinematic story:

* Emotional intro sequences
* Smooth transitions
* Floating balloon animations
* Interactive surprise elements
* Elegant typography
* Memory-inspired visual storytelling
* Heartfelt quotes and messages

The project focuses on emotion-driven UI/UX combined with modern frontend engineering.

---

# 🧠 Design Philosophy

> “A mother’s love deserves more than a static webpage.”

This project combines:

* Cinematic storytelling
* Motion design
* Emotional interaction
* Minimal yet immersive aesthetics
* Human-centered experience

Every animation and transition was intentionally crafted to feel soft, emotional, and meaningful.
